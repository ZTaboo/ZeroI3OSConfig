## polybar使用第三方插件

## Fcitx5

> 搜狗输入法依赖fcitx,但在使用中各种问题，索性使用fcitx5了，先安装的ubuntu再安装的i3wm,可以完整安装中文，如果不是这样需要手动安装中文字体
[fcitx5-nord](https://github.com/tonyfettes/fcitx5-nord)

## picom（渐入渐出透明效果）

> 直接apt install picom即可，配置文件放在~/.config/picom/picom.conf
